# C# Fundamentals — WoMakersCode

Studies and exercises from the WoMakersCode C# introductory track on Microsoft Learn.

## Modules

| Module | Topic | Concepts |
|--------|-------|----------|
| 01 | Write your first C# code | Console.WriteLine, syntax, string literals |
| 02 | Literals and Variables | string, char, int, decimal, bool, var |
| 03 | Basic Operations | +, -, *, /, %, cast, order of operations |
| 04 | Decision Logic | if, else if, else, &&, \|\|, nested if |
| 05 | Arrays and foreach | array, index, Length, foreach, StartsWith() |
| 06 | Switch-case | switch, case, break, default |
| 07 | for loop | for, break, array iteration, FizzBuzz |
| 08 | do-while and while | do-while, while, continue, break |
